Ejercicio de POO (nivel básico) desarrollado en JAVA, extraído de internet.

Se propone crear una clase Serie con los siguientes atributos:

- Titulo.
- Cantidad de temporadas.
- Información sobre si fue entregada o está diponible (entregada = true si fue entregada, entregada = false si está disponible).
- Género.
- Creador.

Consignas:

1) Crear un array de 5 posiciones y cargar todas las posiciones del array con distintas series.
2) Entregar algunas series.
3) Devolver una serie.
4) Obtener la cantidad de series que se encuentran entregadas.
5) Obtener la serie con mayor cantidad de temporadas (a fines de simplicidad de esta práctica, se contemplará que sólo 1 serie será la que tiene mayor cantidad de temporadas).
